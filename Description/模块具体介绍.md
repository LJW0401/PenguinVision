# 模块具体介绍

## 单目相机模块(ros2_hik_camera)
单目相机模块用于采集RGB图像，作为自动瞄准系统的主要图像，实现对枪口方向的机器人的识别与跟踪。
- 发布信息`/image_raw`
- 单独运行该模块<br>
    在总文件夹打开终端，输入
    ```
    ros2 launch hik_camera hik_camera.launch.py 
    ```

## 可视化
在平时对视觉系统进行调节的时候经常需要用到可视化模块来查看各种信息。
